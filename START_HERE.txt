┌──────────────────────────────────────────────────────────────┐
│ 1️⃣  РЕЗЕРВНАЯ КОПИЯ (если есть старая версия)               │
└──────────────────────────────────────────────────────────────┘
   $ cp -r YourOldSite YourOldSite_backup

┌──────────────────────────────────────────────────────────────┐
│ 2️⃣  КОПИРОВАНИЕ ФАЙЛОВ                                       │
└──────────────────────────────────────────────────────────────┘
   Распакуйте архив и скопируйте папку Site в ваш проект

┌──────────────────────────────────────────────────────────────┐
│ 3️⃣  НАСТРОЙКА .ENV                                           │
└──────────────────────────────────────────────────────────────┘
   Откройте backend/.env и замените:
   
   MONGODB_URI=mongodb+srv://ВАШ_USER:ВАШ_PASS@...
   JWT_SECRET=ваш_секретный_ключ

┌──────────────────────────────────────────────────────────────┐
│ 4️⃣  УСТАНОВКА ЗАВИСИМОСТЕЙ                                   │
└──────────────────────────────────────────────────────────────┘
   $ cd Site
   $ npm install
   $ cd backend && npm install

┌──────────────────────────────────────────────────────────────┐
│ 5️⃣  ЗАПУСК                                                    │
└──────────────────────────────────────────────────────────────┘
   $ npm start
   
   Или:
   $ cd backend
   $ node server.js
